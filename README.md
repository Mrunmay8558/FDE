# Fundamental of Programming Exercises

This repository contains small Python exercises for learning programming fundamentals.

## Repository Structure

```text
fundamental_of_programming/
  exercise_one/
    order_payment_exercise.py
    orderpaymentsolution.py
  exercise_two/
    oop_concepts.py
```

## What Is Included

- `exercise_one`: a data-processing exercise that connects users, orders, and payment details.
- `exercise_two`: beginner-friendly notes and runnable examples for Object-Oriented Programming (OOP) in Python.

## Requirements

- Python 3.10 or newer is recommended.

Check your Python version with:

```bash
python3 --version
```

## How To Run The Files

From the repository root:

```bash
python3 fundamental_of_programming/exercise_one/order_payment_exercise.py
python3 fundamental_of_programming/exercise_one/orderpaymentsolution.py
python3 fundamental_of_programming/exercise_two/oop_concepts.py
```

## Learning Goals

This repo is useful for practicing:

- lists, dictionaries, loops, and conditions
- joining related data manually using IDs
- clean output formatting in Python
- basic Object-Oriented Programming concepts
- reading and understanding beginner-friendly Python examples

## Notes

- Generated Python cache files are ignored through `.gitignore`.
- If you create a local virtual environment, it will also be ignored.
