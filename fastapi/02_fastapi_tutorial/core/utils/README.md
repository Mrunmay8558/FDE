# Utils

This folder contains feature-specific helper code.

Difference between `commons` and `utils`:

- `commons` is for helpers reused across the whole project
- `utils` is for helpers that belong to a specific part of `core`

Current subfolder:

- `email/`: helper code for email sending and email template generation

Beginner note:

- If a helper is strongly tied to one feature, it often belongs in `utils`
